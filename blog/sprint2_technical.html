Margin, Padding & Border

When working with CSS and HTML we often want to specifically place and align elements within our document. It is important to know what the Margin, Padding and Border attributes are, so that you can apply them correctly.

Margin:

The margin is the space around the element. It is a gap that is created to separate the element from surrounding elements. You can specify the size of the margin on each side of the element, or you can specify the size of the margin on only one side of the element, depending on what effect you are trying to achieve.

Padding:

The Padding is the space within the block that tells elements how far apart they should be from the border. Much like the Margin, you can specify which side and sizes to use for padding. This can be helpful for aligning images or text elements inside a particular section on the page, like a grid.

Border:

The border exists on the edges of the block. It can be specified to draw attention to itself and stand out from the background. This is useful for separating elements, especially in the NAV. With the border you can specify the Style, Width, and even create rounded corners using the border-radius property. You can also adjust the colour to make it stand out.
